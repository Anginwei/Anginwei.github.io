default.css

简易的浏览器样式重置，部分效果引用自bootstrap

简介

default.css只做最少的修改，使常见页面元素有统一的基本样式。一些基础默认样式参考了bootstrap，修改内容主要包括全局body样式、文字排版、表单、表格。

这个页面为了展示最原始的效果，不进行任何配色美化，源码从这里获取。

default.css源码用LESS编写，推荐自行修改基础样式，文件结构如下：

文件 用途
default.less 引用其他文件的索引，使用源码时，只用编译这个文件
normalize.less 有名的normalize库，修正了很多控件的表现
core.less default.css核心文件，所有的改动都在这
variable.less 用于保存LESS变量，可以从这里修改default.css默认值，里面有详细注释
mixin.less 存放了一下常用的LESS的混合类，如果你习惯用LESS写CSS样式的话，这个文件会很有用，可以里面增加需要的LESS混合
